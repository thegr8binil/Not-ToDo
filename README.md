# Not-ToDo

A simple and interactive To-Do list application built with **Next.js** and **Framer Motion** for smooth animations. This app allows users to manage their daily tasks, mark them as completed, and remove them from the list with a fluid animation effect.

---

## 🚀 Features

- **Add tasks** with a specified duration.
- **Toggle tasks** as completed or pending.
- **Remove tasks** with smooth exit animations.
- Uses **Framer Motion** for animations and transitions.
- **Beautiful** and responsive design with **Tailwind CSS**.

---

## 🛠 Technologies Used

- **Next.js**: A React framework for building server-side rendered and static web applications.
- **Framer Motion**: A powerful library for creating animations in React.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs.
